
# Reanimated TabBar

A Reanimated TabBar based on expo router

![video.gif](assets/video.gif)


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
